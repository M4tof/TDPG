# TDPG - Tower Defense Procedural Generation

## Project Status: Archive & Migration Note
**EN:** This repository is a port/migration from an internal GitLab instance. Due to the migration process, the contributor statistics on GitHub may not accurately reflect the original development history. This project was co-authored by **four contributors**.

**PL:** To repozytorium jest migracją z wewnętrznej instancji GitLab. Ze względu na proces portowania, statystyki współtwórców na GitHubie mogą nie odzwierciedlać w pełni oryginalnej historii zmian. Projekt był współtworzony przez **czterech autorów**.

## Description | Opis
**EN:** Engineering project focused on creating a procedural generation system for Tower Defense games. This repository contains the full Unity project, including the source code for the TDPG library and a functional tech demo.

**PL:** Projekt inżynierski mający na celu stworzenie systemu proceduralnej generacji do gier typu Tower Defense. Repozytorium zawiera pełny projekt Unity, w tym kod źródłowy biblioteki TDPG oraz demo technologiczne.

## 🇺🇸 English Version

### Project Structure
- **`tower_defence_inz/`**: The complete Unity project source code for the tech demo.
  - **`Assets/TDPG`**: This specific folder contains the core source code for the procedural generation package.
- **`dokumentacja/`**: Project documentation and history.
  - **`dokumentacja/materiały_ze_spotkań`**: Archive of early design documents and meeting notes.
  - **`dokumentacja/wstępna`**: Pre-sprint design documents (slightly outdated).
  - **`dokumentacja/System_proceduralnego_generowania_funkcjonalności_w_grach_typu___tower_defense.pdf`**: The final engineering thesis containing the official technical documentation.

### Downloads (Releases)
The following pre-built files are available in the [**Releases**](https://github.com/M4tof/TDPG/releases) section:
1. **TDPG.unitypackage**: The isolated library ready for import into other Unity projects.
2. **Gra.zip**: Compiled tech demo. 
   - Extract and launch `tower defence inz.exe`.
